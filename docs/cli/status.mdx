---
title: glosso status
description: Show the active wallet mode, address, and balance.
---

## Usage

```bash
glosso status [options]
```

## Options

| Flag | Type | Default | Description |
|---|---|---|---|
| `--network` | `devnet \| mainnet-beta` | from `.env` | Override network |
| `--index` | number | `0` | Sub-wallet index to query |
